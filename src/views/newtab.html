FAQ

How can I use the keyboard shortcuts?

Just like the old Firefox Speed Dial extension, you can use `Ctrl+<number>` to open in the same tab and `Alt+<number>` to open in a new tab.

The keyboard shortcuts are not working. What am I doing wrong?

Unfortunately, new web browsers only allow extensions to have arbitrary non-fixed keyboard shortcuts when a web page is focused.

I find the keyboard shortcuts annoying. How can I disable them?

Check the checkbox in the top left of the new tab and the shortcuts will only work if `Shift` is key is pressed.

What are the pages/thumbnails I can import?

Currently, you can import from one of the following:
1. Firefox's old Speed Dial extension.
2. The same extension (Speed Dial Quantum).

And support for others may be added based on how many users can benefit.

The new tab page sometimes shows the browser's default one. What's going on?

This can happen when the extension is updated and I need to do some research to see if this can be avoided.

What are the supported web browsers?

Currently, the Firefox version before quantum (56) and above and any reasonably new version of Google Chrome/Chromium and support for Microsoft Edge & Opera is planned.

What are some of the planned changes/features?

The only visible planned change/feature, at the moment, is using a global single-shortcut. For example, `Ctrl+Shift+X` (similar to 1Password) that would open the extension popup and you can take it from there. Because many users are not used to the current behavior and it also requires access to all web pages.
